Bootstrap Dropdown Examples & Explanations

1. Basic Dropdown

This is the simplest form of a Bootstrap dropdown. Click the button to see the dropdown menu appear.

  • Action
  • Another action
  • Something else here
<div class="dropdown"> <button class="btn btn-secondary dropdown-toggle" type="button" id="basicDropdown" data-bs-toggle="dropdown" aria-expanded="false"> Basic Dropdown </button> <ul class="dropdown-menu" aria-labelledby="basicDropdown"> <li><a class="dropdown-item" href="#">Action</a></li> <li><a class="dropdown-item" href="#">Another action</a></li> <li><a class="dropdown-item" href="#">Something else here</a></li> </ul> </div>

Key Components Explained:

  • .dropdown - Container class that wraps the entire dropdown
  • .dropdown-toggle - Makes the button look like a dropdown trigger
  • data-bs-toggle="dropdown" - Essential attribute that enables dropdown functionality
  • .dropdown-menu - The actual dropdown content container
  • .dropdown-item - Individual items within the dropdown

  • aria-labelledby - Links button to menu
ðŸ’¡ Pro Tips:
  • Always include data-bs-toggle="dropdown" for functionality
  • Use aria-labelledby to connect button and menu for accessibility
  • Bootstrap 5 uses data-bs-toggle (Bootstrap 4 used data-toggle)
  • Dropdowns are click-activated, not hover-activated
  • Use .btn-group for split button dropdowns
